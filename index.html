HANGMAN

This is the hangman game. The computer has randomly selected a word. The category is hard words so you will need to think carefully in order to beat the computer. When the game starts underscores will appear on the screen. The number of underscores will tell you how many letters are in the word. You will select a letter from the buttons at the bottom of the screen to see if you can guess the word. Once you select a letter you will not be allowed to select it again. If you guess the word before you miss 7 times then you win. If you fail to guess a correct letter 7 times you lose.
